w_RtpNumObus: number
  • Defined in packages/rtp/src/codec/av1.ts:75

RtpNumObus two bit field that describes the number of OBU elements in the packet. This field MUST be set equal to 0 or equal to the number of OBU elements contained in the packet. If set to 0, each OBU element MUST be preceded by a length field.
