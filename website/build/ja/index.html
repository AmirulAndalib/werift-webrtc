Pure WebRTC implementation of the WebRTC

Get Started

ピュアTypeScript

Node.jsの標準ライブラリとjs製のライブラリのみで構成されているため、Node.jsさえインストールされていれば動作します。ネイティブモジュールに一切依存していないので、node-gypなどに煩わされることがありません。

シンプルなAPI

ブラウザのWebRTC APIに近いAPI設計となっており、学習コストが抑えられています。

学習用に最適

コードベースがとても小さく、WebRTCの全てのプロトコルスタックが一つのリポジトリに収まっています。ビルドも非常に簡単なので自分で簡単に機能を追加したり修正したりできます。
